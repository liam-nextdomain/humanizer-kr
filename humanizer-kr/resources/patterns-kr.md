### AI 반복 동사·형용사 다양화 참고표 (Park & Kim, 2025)

AI는 특정 동사·형용사를 반복 사용하는 경향이 있다. 동일 표현이 3회 이상 반복되면 아래 대체어를 활용한다.

| AI 반복 동사 | 대체어 |
|-------------|--------|
| 활용하다 | 쓰다, 도입하다, 적용하다, 끌어오다, 접목하다 |
| 기여하다 | 이바지하다, 뒷받침하다, 힘을 보태다, 보탬이 되다 |
| 미치다 (영향을) | 작용하다, 좌우하다, 끼치다, 파급되다 |
| 제공하다 | 내놓다, 건네다, 마련하다, 내어주다 |
| 향상시키다 | 높이다, 끌어올리다, 개선하다, 나아지게 하다 |

| AI 반복 형용사 | 대체어 |
|---------------|--------|
| 다양한 | 여러, 갖가지, 폭넓은, 다채로운 |
| 중요한 | 핵심적인, 결정적인, 빼놓을 수 없는, 무게 있는 |
| 효과적인 | 실질적인, 유용한, 알찬, 성과를 내는 |
| 긍정적인 | 바람직한, 고무적인, 희망적인 |
| 부정적인 | 우려스러운, 해로운, 좋지 않은 |

## 4. 고빈도 표현 대체/삭제 전략

### 접속어 처리 (삭제 우선 원칙)

| 유형 | 접속어 | 처리 방법 | 예시 |
|------|--------|----------|------|
| 순접·보충 | 또한 | 흐름이 자연스러우면 삭제 | "또한 중요하다" → "중요하다" |
| 순접·보충 | 뿐만 아니라 | 삭제 후 문장 통합 | "A다. 뿐만 아니라 B다" → "A이고 B다" |
| 순접·보충 | 이를 통해 | 삭제 또는 직접 동사로 | "이를 통해 개선된다" → "개선된다" |
| 순접·보충 | 더불어 | 삭제 | → 삭제 후 문장 연결 |
| 순접·보충 | 나아가 | 삭제 또는 새 문장 시작 | → 대부분 삭제 |
| 인과·결과 | 따라서 | 인과가 명확하면 삭제 | "중요하다. 따라서 필요하다" → "중요하므로 필요하다" |
| 역접·대조 | 그러나 | 대조가 약하면 삭제 | → 내용 대조가 강할 때만 유지 |
| 역접·대조 | 하지만 | 대조가 약하면 삭제 | → "그러나"와 동일 |
| 요약·정리 | 이처럼 | 거의 항상 삭제 | → "이처럼 중요하다" → "중요하다" |
| 요약·정리 | 결과적으로 | 삭제 후 문장 직접 서술 | → 대부분 삭제 |
| 인과·결과 | 그러므로 | "따라서"와 동일 처리 | |

### 예시·강조형 접속사 처리 (AI 과다 사용 유형 — Park & Kim, 2025)

AI 텍스트에서 가장 두드러지는 과잉 유형. 같은 글 안에서 3회 이상 등장하면 적극 개입한다.

| 접속어 | 처리 방법 | 예시 |
|--------|----------|------|
| 예를 들어 | 빈도 높으면 일부를 구체적 사례 직접 서술로 대체 | "예를 들어 A가 있다" → "A의 경우를 보면" 또는 삭제 후 사례 직접 제시 |
| 특히 | 무엇이 특별한지 구체화하거나 삭제 | "특히 중요하다" → 왜 중요한지 직접 서술 |
| 예컨대 | "예를 들어"와 교번 사용 시 하나 삭제 | 둘 다 있으면 하나만 남기기 |
| 다시 말해 | 앞 문장이 충분히 명확하면 삭제 | 앞 문장을 더 명확하게 고쳐 반복 제거 |
| 사실은 | 강조 의도가 약하면 삭제 | "사실은 A다" → "A다" |

> **조건형 접속사 활용 팁**: AI 글에서 거의 부재하는 조건형(한다면, ~라면, 만약)을 적절히 삽입하면 인간적 사고 패턴이 드러난다. "예를 들어 A가 발생한다" → "A가 발생한다면" 등으로 전환 가능.

**연속 접속어 규칙**: 3문장 이상 접속어로 시작하면 반드시 1개 이상 삭제.

### 강조어 처리 (구체적 사실로 대체)

| 강조어 | 처리 방법 |
|--------|----------|
| 매우 중요하다 | 구체적 이유로 대체: "이 단계를 건너뛰면 X가 작동하지 않는다" |
| 특히 | 무엇이 특히인지 구체화하거나 삭제 |
| 상당히 | 수치나 구체적 표현으로 대체 |
| 크게 향상되었다 | "X% 향상", "두 배 빨라졌다" 등 구체화 |
| 효과적으로 | 무엇에 효과적인지 서술하거나 삭제 |
| 적극적으로 | 구체적 행동으로 대체 |
| 지속적으로 | 빈도나 기간으로 구체화 또는 삭제 |

### 틀에 박힌 수식어 처리

| 표현 | 처리 방법 |
|------|----------|
| 다양한 측면에서 | 어떤 측면인지 명시하거나 삭제 |
| 중요한 역할을 한다 | 어떤 역할인지 직접 서술 |
| 긍정적인 영향을 미친다 | 구체적으로 어떤 영향인지 서술 |
| 부정적인 영향을 미친다 | 동일 |
| 효과적으로 활용 | 어떻게 활용하는지 직접 서술 |

## 6. 에세이 vs 논문 처리 전체 비교표

| 패턴 | 에세이 처리 | 논문/보고서 처리 |
|------|------------|----------------|
| **쉼표 과다** | 적극 제거, 문장 분리 또는 접속어 생략 | 영어식(접속어 앞, 주어-서술어 사이)만 제거 |
| **명사 중심 구조** | 동사/구어체 전환 ("~이 필요하다" → "~해야 한다") | 핵심 명사는 유지, ~의 ~의 중첩만 해소 |
| **AI 고빈도 표현** | 접속어 삭제 우선, 수식어 구체화 | 삭제 또는 정확한 대체어로 교환 |
| **3의 법칙 나열** | 해체, 가장 중요한 항목만 남기기 | 나열 필요 시 유지, 번호 표지는 상황에 따라 |
| **완결형 틀 구조** | 서론 상투어 제거, 결론 상투어 제거, 내용으로 바로 진입 | 서론/결론 구조는 유지, 공허한 상투어만 제거 |
| **접속어 남발** | 연속 접속어 적극 삭제 | 과도한 것만 제거 |
| **의존명사 띄어쓰기** | 붙여쓰기 허용 (~는것, ~할때) | 표준 띄어쓰기 엄격 유지 |
| **보조용언 띄어쓰기** | 붙여쓰기 허용 (되어있다, 해주다) | 표준 띄어쓰기 엄격 유지 |
| **감정/개성 부재** | 필자 반응, 주관 표현 추가 가능 | 적용 안 함. 객관성 유지 |
| **커뮤니케이션 아티팩트** | 완전 제거 | 완전 제거 |

## 7. AI 글쓰기 감사 체크리스트

### 에세이/블로그 감사 항목 (재작성 후 확인)

- [ ] 전체 문장 중 쉼표 포함 비율이 30% 이하인가?
- [ ] "뿐만 아니라", "또한", "따라서", "이처럼"이 연속 3문장 이상 없는가?
- [ ] "중요한 역할", "다양한 측면", "효과적으로 활용"이 사라졌는가?
- [ ] 서론에 "~에 대해 살펴보겠다", "~을 알아보고자 한다"가 없는가?
- [ ] 결론에 "~이 중요하다는 것을 알 수 있다", "~해야 할 것이다"가 없는가?
- [ ] 필자의 목소리 또는 반응이 최소 1회 이상 드러나는가?
- [ ] 원문의 종결 어미 tier(높임말/반말)가 재작성 후에도 유지되었는가?
- [ ] 문장 길이가 단문과 복문이 적절히 혼합되어 있는가?
- [ ] 인사말, 이모지, 굵은 글씨 헤더가 제거되었는가?
- [ ] 명사구 중첩(~의 ~의 ~)이 동사절로 전환되었는가?
- [ ] 개조식 문장(단문 나열·번호 세분화)이 서술형으로 전환되었는가?
- [ ] 동일 동사·형용사가 3회 이상 반복되지 않는가?
- [ ] 예시·강조형 접속사(예를 들어, 특히, 예컨대)가 과도하지 않은가?

### 논문/보고서 감사 항목 (재작성 후 확인)

- [ ] 나열 이외 용도의 쉼표가 대폭 줄었는가?
- [ ] "다양한", "효과적인", "중요한"이 내용 없이 반복되지 않는가?
- [ ] 전문 용어는 일관되게 사용되었는가? (동의어 혼용 없음)
- [ ] 서론이 "~에 대해 고찰하고자 한다"로 시작하지 않는가?
- [ ] 결론이 "~이 중요함을 알 수 있다"로만 끝나지 않는가?
- [ ] 표준 띄어쓰기가 유지되었는가?
- [ ] 객관적 서술 어조가 유지되었는가?
- [ ] 인사말, 이모지가 제거되었는가?
- [ ] 개조식 구조가 논문 형식에 적합한 수준으로 제한되었는가?
- [ ] 동일 표현의 과도한 반복이 해소되었는가?

## 8. 완결형 틀 구조 — 상투어 목록

### 서론 상투어 (제거 대상)

### 개조식 문장 경향 (정성 분석)

- AI 활용 글: 서술형 대신 단문 나열·요약적 진술(개조식) 비율 현저히 높음
- 목차 세분화 패턴: Ⅰ → 1 → 1) → (1) → ① 등 다단계 구조화
- 인간의 글: 자신의 논지를 서술형으로 전개하는 글이 다수
- AI 활용 글: 소주제를 나누고 요약하는 획일화된 형태 다수 발견

## 10. 한국어 어미 다양화 참고

LLM은 특정 어미 패턴을 반복한다. 재작성 시 어미를 다양하게 활용하여 POS 다양성을 높인다.

### LLM 과빈도 어미

| LLM 선호 어미 | 자연스러운 대안 |
|--------------|--------------|
| ~하며 (동시적 병렬) | ~하고, ~해서, ~하는데 |
| ~하여 (원인/수단) | ~해서, ~했더니, ~하니까 |
| ~하고 있다 (진행) | ~하는 중이다, ~하는 상황이다, ~하는 중 |
| ~할 수 있다 (가능) | ~된다, ~하게 된다, ~가능하다 |
| ~해야 한다 (당위) | ~해야 하는 상황이다, ~안 할 수가 없다 |
| ~됨으로써 | ~되어서, ~되면서 |
| ~함으로써 | ~해서, ~하면서 |
| ~에 따라 | ~에 맞게, ~을 보면, ~이니까 |
