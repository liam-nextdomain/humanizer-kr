# Output Format Templates

Templates for each phase of the humanizer-kr workflow. SKILL.md references this file at Steps 2.5, 3.5, 4, and 5.5.

---

## Detection Report (Step 2.5)

```text
**[패턴 감지 결과]**

**[A] 구두점 패턴**
- P1 (쉼표 남용): N건 — 예시: "..."

**[B] 구조/어순 패턴**
- P3 (삼단 나열·개조식): N건 — 예시: "..."

**[C] 어휘/표현 패턴**
- P5 (AI 상투 표현): N건 — 예시: "..."

**[D] 띄어쓰기 패턴** *(에세이 전용)*
- P7 (의존명사 띄어쓰기): N건 — 예시: "..."

**[E] 소통 패턴**
- P10 (소통 부산물): N건 — 예시: "..."

전체 수정을 진행할까요? 특정 유형만 선택하실 수도 있고, 거부하실 수도 있습니다.
```

---

## Draft + Change Brief (Steps 3 & 3.5)

```text
**1차 재작성본 (Draft)**
[rewritten text]

**[1차 수정 브리핑]**
- [A] ...
- [C] ...
```

---

## Voice Consultation (Step 4 — Essay Only)

```text
**[목소리 협의 — 에세이 전용]**

**[V1]** 원문 문장 발췌
→ 어떤 방향이 가장 가깝나요?
1. (동의/긍정) 예시
2. (회의/비판) 예시
3. (개인 경험) 예시
4. (열린 물음) 예시
5. (직접 입력) 원하시는 관점을 말씀해 주세요.
```

---

## Re-Validation Report (Step 5)

```text
**[재검증 결과]**
- P_X (설명): N건 — 예시: "..."
추가 수정을 진행할까요?
```

---

## Final Output + Brief (Step 5.5)

```text
**최종본**
[final rewritten text]

**[최종 수정 브리핑]**
- [category] ...
```
